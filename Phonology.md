### Phonemes

|                   | Bilabial | Labio-dental | Alveolar     | Palatal      | Velar   | Glottal |
| ----------------- |:--------:|:------------:|:------------:|:------------:|:-------:|:-------:|
| Plosive           | p        |              |    t, tʲ         |              | k       |         |
| Nasal             | m        |              |    n         |      ɲ       |         |         |
| Trill             |          |              |    r         |              |         |         |
| Fricative         |          |     v, f     |     s, ʃ, sʲ |              |         | h       |
| Lateral           |          |              |    l         |      ʎ       |         |         |
| Approximant       |          |              |              |      j       |         |         |

|       | Front | Central | Back  |
| ----- |:-----:|:-------:|:-----:|
| Close |   i   |         |   u   |
| Mid   |   e   |         |   o   |
| Open  |       |    a    |       |


Vowels can be short as above or long, which simply holds pronunciation longer (for example aː iː uː eː oː)
